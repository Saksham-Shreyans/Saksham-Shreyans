## 🚀 Featured Projects

### 🛡️ [QuantHunt — Post-Quantum Cryptography Scanner](https://github.com/Saksham-Shreyans/Quanthunt)
> **The only student-built PQC readiness intelligence platform of its kind**

A full-stack cybersecurity scanning engine that audits, maps, and secures digital infrastructure against next-generation quantum threats.

**In under 180 seconds, QuantHunt:**
- 🔍 Performs live TLS handshake scans across **all discovered subdomains**
- 📜 Passively mines Certificate Transparency logs — no target contact
- ⚛️ Evaluates every cryptographic primitive: key exchange, auth algorithm, TLS version, cert signature, symmetric cipher
- 📊 Scores each asset on a **0–100 HNDL risk scale** using a 5-dimension weighted formula
- 🔗 Generates a **blockchain-backed audit trail** (SHA-256 PoW per scan, tamper-evident)
- 📋 Exports **CycloneDX 1.6 CBOM** with NIST FIPS 203/204/205 flags & CERT-IN Annexure-A mapping
- ⚡ Fleet mode: **batch scan 300+ domains** in a single API call

**Also built:** PQC TLS Latency Studio — a 3-wire PASS/HYBRID/FAIL simulation mathematically modeling real TTFB degradation cost of quantum-safe migration.

**Tech Stack:** Backend — FastAPI 0.116 + SQLAlchemy 2.0 + SQLite (dual-DB) + Celery + ReportLab · Frontend — React 18 + Recharts 2.12 + HTML5 Canvas API · Security — Domain allow-list enforcement, parameterized SQL, CORS hardening, scan idempotency, server-side certificate gating
